Setting Up PS2 BIOS in PCSX2 (PC — Windows / Mac / Linux)

PCSX2 is the gold standard for PS2 emulation on desktop. The setup takes about 5 minutes once you have the BIOS file in hand.

1
Download and extract the BIOS file
Get your BIOS from allps2bios.com. The file usually comes as a ZIP archive. Extract it — you should see files ending in .bin, .erom, .nvm, or .rom. Keep them all together in one folder.
Don't rename the files. PCSX2 looks for them by exact filename.
2
Place BIOS files in the correct folder
Move all the extracted BIOS files into PCSX2's BIOS directory.
Windows: C:\Users\YourName\Documents\PCSX2\bios\
Mac / Linux: ~/Documents/PCSX2/bios/
3
Open PCSX2 and go to Settings
Launch PCSX2. Click Settings in the top menu, then select BIOS from the left sidebar.
4
Set the BIOS directory path
If PCSX2 didn't auto-detect your BIOS folder, click Browse and navigate to the folder where you placed the BIOS files. PCSX2 will scan the folder and list any valid BIOS files it finds.
5
Select your BIOS version
You'll see a list of detected BIOS files with their region labels. Click on the one that matches your games — USA for NTSC-U, Europe for PAL, Japan for NTSC-J.
6
Apply and test
Click Apply or OK to save. Then load a game via File → Boot ISO. If the PS2 boot screen appears with the Sony logo and memory card animation, your BIOS is working perfectly.

Setting Up PS2 BIOS in AetherSX2 (Android)

AetherSX2 setup on Android is straightforward but the file management step trips up a lot of people. Here's exactly how to do it.

Note

AetherSX2 development has slowed in recent years. Some users switch to NetherSX2 (a community fork). The BIOS setup process is identical for both.

1
Transfer the BIOS file to your Android device
Copy the BIOS file from your PC via USB, or download it directly on the phone from allps2bios.com. Create a dedicated folder for it — something like Internal Storage / PS2 / BIOS /.
2
Install AetherSX2 and open it
Install from the Play Store or APK. On first launch, the setup wizard will ask for your BIOS folder. Tap Select when it prompts for the BIOS directory.
3
Grant storage permissions
AetherSX2 needs permission to read your files. Accept the storage access prompt when it appears. Without this, it won't be able to find the BIOS even if you've set the path correctly.
On Android 11+, you may need to grant "All Files Access" permission in Settings → Apps → AetherSX2.
4
Navigate to and select the BIOS folder
Use the file picker to navigate to your PS2 / BIOS / folder. Tap the folder (not the file inside it) and confirm. AetherSX2 scans the folder and detects the BIOS automatically.
5
Complete setup and load a game
Follow the remaining setup wizard steps for game directories. Once done, add a PS2 ISO and tap it to launch. A successful boot shows the PS2 startup screen before the game loads.

Common Errors and How to Fix Them

ErrorLikely CauseFix
BIOS Not Found Wrong folder path or files in a subfolder Make sure BIOS files are directly inside the bios folder, not in a nested subfolder
Black Screen Region mismatch between BIOS and game Switch to a BIOS that matches your game's region
Invalid BIOS Corrupted or wrong file type Re-download from allps2bios.com and extract fresh
No BIOS Listed Emulator can't read the folder Check storage permissions on Android; recheck directory path on PC
Game Crashes at Boot Wrong emulator version or settings Update PCSX2/AetherSX2 to the latest build; reset to default settings
Audio/Speed Issues PAL game running on NTSC BIOS Switch to the PAL BIOS for European games

Frequently Asked Questions

Where exactly should I put the PS2 BIOS files?
On PC (PCSX2): the bios subfolder inside your PCSX2 documents folder. On Android (AetherSX2): any folder you choose — just point the app to it during setup.
Why does PCSX2 say "no BIOS found" even after I placed the files?
Usually means the files are in a subfolder inside the bios folder, or the path is pointing to the wrong directory. Check that the .bin files are directly inside the bios folder, not nested further.
Does AetherSX2 need a different BIOS than PCSX2?
No. The exact same BIOS file works for both. You can copy the file from your PC to your phone and it works fine.
Which BIOS region should I use for most games?
USA (NTSC-U) for most English games. For details on every region, check the region comparison guide.
Where can I download the PS2 BIOS for free?
All region files are available at allps2bios.com.
